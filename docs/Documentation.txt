STUDENT INFORMATION SYSTEM

---

## 📌 1. Project Description

This project is a console-based Student Information System developed using Java.
The system allows users to manage student records efficiently by performing
operations such as adding, viewing, searching, updating, and deleting student data.

It is designed using Object-Oriented Programming principles like encapsulation,
where student data is managed through private variables and accessed using
getter and setter methods.

---

## 📌 2. Features

1. Add Student

   * Allows user to enter student details such as ID, name, age, grade, and contact.

2. View All Students

   * Displays all student records in a structured tabular format.

3. Search Student

   * Searches for a student using their unique Student ID.

4. Update Student

   * Updates existing student details like name, age, grade, and contact.

5. Delete Student

   * Removes a student record using Student ID.

---

## 📌 3. Technologies Used

* Java (Core Java)
* ArrayList (Dynamic data storage)
* Scanner (User input handling)

---

## 📌 4. How to Run the Program

1. Open terminal or command prompt

2. Navigate to project directory

3. Compile the program:
   javac src/StudentInformationSystem.java

4. Run the program:
   java -cp src StudentInformationSystem

* Data is lost after program termination
* Console-based system (no graphical user interface)
